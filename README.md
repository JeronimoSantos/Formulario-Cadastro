<h1>Formulario Responsivo - HTML e CSS</h1>
<p>Um pequeno projeto que desidir fazer como uma forma de praticar e relembrar conseitos e me manter atualizado.</p>

<h2>Objetivo Principal</h2>
<p>Criar um formulario de cadastro responsivo com uma organização de pastas, inputs diferentes mais uma imagem ilustrativa ao lado.</p>

<h2>Tecnologias Utilizadas</h2>
<ul>
    <li>HTML</li>
    <li>CSS</li>
</ul>

<h2>Melhorias</h2>
<ul>

</ul>
